# IronPython-example

Bu repo, bir Windows Forms uygulamasından IronPython kullanarak harici Python scripti çalıştırmayı gösteren örnek bir projedir.

## Ne Yapar

- IronPython motorunu başlatır
- Python script dosyasını C# içinden çalıştırır
- Python çalışma yollarını uygulama içinden tanımlar

## Teknolojiler

- C#
- Windows Forms
- IronPython
- Python

## Notlar

- Kod içinde yerel dizinlere bağlı örnek yollar bulunur; farklı ortamda çalıştırmadan önce güncellenmesi gerekir
